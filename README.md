# 📦 E-Commerce Supply Chain Analytics with Demand Forecasting

An end-to-end data analytics project built using the Olist Brazilian E-Commerce dataset. This project combines SQL, Python, Machine Learning, Demand Forecasting, Excel, and Power BI to analyze supply chain performance, delivery operations, customer behavior, and future revenue trends.

---

## 🚀 Tech Stack

- Python (Pandas, NumPy, Matplotlib, Seaborn)
- MySQL Workbench
- Scikit-learn
- Prophet
- Microsoft Excel
- Power BI

## ⭐ Key Features

- Cleaned and prepared the Olist e-commerce dataset.
- Performed SQL-based business and supply chain analysis.
- Conducted exploratory data analysis (EDA) using Python.
- Built a Random Forest model to predict late deliveries.
- Forecasted future revenue using Prophet.
- Designed an interactive Power BI dashboard for business insights.
